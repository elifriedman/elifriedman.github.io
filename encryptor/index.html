This tool uses modern authenticated encryption (AES‑256‑GCM + Argon2id) to protect your data:

  • Key Derivation: The password is turned into an encryption key using Argon2id (memory‑hard, GPU‑resistant). Fallback if Argon2id is not available: PBKDF2 (200k iterations).
  • Encryption: AES‑256‑GCM, which provides both confidentiality and integrity (tamper detection).
  • Randomness: Each file uses a fresh 128‑bit salt and 96‑bit nonce.
  • Authenticated Metadata: File header (version, KDF params, salt, nonce) is included as authenticated data.
  • File Format: MAGIC | header length | JSON header | ciphertext

Security depends on a strong password and a trusted browser environment.
